# Atividade Prática - Biblioteca React

## Arquivos do projeto

- `index.html` → exemplo com JavaScript tradicional
- `react.html` → atividade em React usando CDN

## Respostas

1. **O React manipula o DOM diretamente?**  
   Não, ele usa o Virtual DOM para ser mais eficiente.

2. **Qual a vantagem de componentes?**  
   Eles ajudam a reutilizar código e organizar melhor a aplicação.

3. **O que substitui o for no React?**  
   O método `map()` para renderizar listas.
